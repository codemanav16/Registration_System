# Registration System

##  Overview
The Registration Management System is a web-based application built using PHP and MySQL, designed to manage user registration data efficiently. The project runs on the XAMPP server and demonstrates backend database connectivity..

##  Technologies Used
- PHP
- MySQL
- XAMPP Server
- bootstrap

##  Features
- User registration form
- Data storage using MySQL database
- View registered users
- Simple and clean UI

##  Setup Instructions
1. Install **XAMPP**
2. Start **Apache** and **MySQL**
3. Clone or download this repository
4. Move the project folder to:C:\xampp\htdocs\registration


5. 5. Create a database in **phpMyAdmin**
6. Import the provided SQL file
7. Update database credentials in PHP files if required
8. Open browser and run:http://localhost/phpmyadmin

-login.php
